# Student Attendance Management System
This project is a simple Python-based application used to manage student attendance digitally.
## About the Project
The system helps teachers record and manage student attendance efficiently.
It reduces manual work and keeps attendance records organized.
## Features
- Add student details
- Mark attendance
- View attendance records
- Simple and easy to use
## Technologies Used
- Python
- SQLite
- Git
## Installation
1. Clone the repository
2. Install dependencies
3. Run the project
## Usage
Run:
python main.py
